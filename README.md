# HotelRevAI-Infosys-Internship-project

•	Working on “HotelRevAI – AI Driven Revenue Analysis for Hotels”, analyzed real-world hotel datasets to evaluate key KPIs such as ADR (Average Daily Rate), RevPAR (Revenue per Available Room), and Occupancy Rate to identify revenue trends and performance patterns.

•	Performed ETL and exploratory data analysis (EDA) using Excel and Python to clean, transform, and extract actionable insights for hotel pricing and revenue optimization.

•	Developing interactive dashboards in Power BI / Excel and applying AI-based forecasting techniques to visualize hotel performance and generate data-driven business insights
